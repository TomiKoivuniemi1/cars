# Cars Viewer

## Overview

Cars Viewer is a web application that displays information about different car models, manufacturers, and specifications using an external Cars API. The project focuses on client-server interaction and presenting data in a clear, user-friendly format.

## Install and execute

* First run the api server
1. open terminal and navigate to the api folder
2. npm install
3. make run

* Then run the program and enter the site
1. open (another) terminal and navigate back to cars folder
2. go run .
3. open link http://localhost:8080


## Requirements

* A frontend to fetch and display car data from an API
* UI components (tables/cards)
* Uuser-triggered actions to fetch additional car details
* Go backend to handle API requests and responses

## Technologies Used

* **Go** – backend server and API handling
* **JavaScript (Node.js / frontend JS)** – client-side logic and interaction
* **HTML / CSS** – user interface
* **REST API** – data fetching and communication
* **JSON** – data format for API responses

## Features

* Display list of cars with basic information
* Fetch detailed specs on user interaction
* Client-server communication via API calls

## Learning goals

* Working with external APIs and JSON data
* Building full-stack applications (Go + JS)
* Implementing dynamic UI updates
* Handling client-server communication effectively
